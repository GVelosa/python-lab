# Reusable Cards with Flet

A small study project focused on building **reusable UI components** in Python using the **Flet** library.  
The goal is to practice creating modular card components that can be reused to represent different functionalities inside an application.

> [!NOTE]  
> This project was created as a **learning exercise** to explore component-based UI design with Flet.

## Features
* Reusable card component structure.
* Image, title, and description layout.
* Click interaction on cards.
* Dynamic styling (e.g., background color changes).
* Modular component design for easier scalability.

## Technologies Used

| Technology | Description |
| :--- | :--- |
| [Python](https://www.python.org/) | Programming language used for the application logic. |
| [Flet](https://flet.dev/) | UI framework used to build the interface components. |

## Purpose of the Project

This repository focuses on learning how to:

* Create **reusable UI components**.
* Structure UI code in a **modular way**.
* Reduce duplication by encapsulating UI elements into components.
* Prepare the foundation for larger applications built with Flet.

## Example Component Structure

The main idea of this project is to encapsulate the card layout into a reusable component:

- Image
- Title
- Description
- Container styling
- Interaction behavior

This allows the same card to be reused to represent different tools or features in future applications.
