## Canal (Basic Class)

A simple example of a class representing a channel.

### Concepts covered:
- Class creation
- Attributes
- Methods
- Object instantiation

### Features:
- Stores name, description, and number of subscribers
- Allows incrementing subscribers

## Employee System

This is the only code in this folder that was fully created by me, but still inspired by the video.

### Concepts covered:
- Inheritance
- Encapsulation
- Use of `@property`
- Basic relationship between objects

### Features:
- `Employee` class with basic attributes
- `EmployeeDirector` class that manages a team
- Prevents duplicate team members

---

## Channel System (Complete Example)

A more complete OOP system simulating a content platform (like YouTube).

### Concepts covered:
- Composition
- Inheritance
- Lists of objects
- Object interaction
- Data encapsulation

### Main classes:
- `Channel`
- `BusinessChannel`
- `Video`
- `Playlist`

### Features:
- Create channels and business channels
- Add/remove playlists
- Post videos
- Track views, likes, dislikes, and comments
- Manage teams (for business channels)
- Organize videos into playlists
