# Factorial Calculator (CLI)

A simple terminal-based factorial calculator built with Python.

The program asks the user to enter a non-negative integer and then calculates its factorial using a loop. The result is displayed directly in the terminal.

---

## Concepts Used

- Variables
- For loops
- Conditional statements (`if`, `else`)
- User input handling
- Basic arithmetic operations
- Basic control flow

---

## How It Works

1. The program asks the user to enter a non-negative integer.
2. If the number is negative, the program displays an error message.
3. If the number is valid, the program calculates the factorial using a loop.
4. The factorial result is printed in the terminal.
